### Requisitos

[Node.js](https://nodejs.org/pt-br/)

[yarn](https://yarnpkg.com/)


## ⚙️ Instalar

Instalar as dependencias:

```
yarn
```

ou

```
npm install
```

## 🔧 Executar os testes

Execução dos testes unitarios:

```
yarn test
```

ou 

```
npm test
```

## 🎈 Utilizar

Executar aplicação na porta 3333:

```
yarn dev
```

ou

```
npm test
```
